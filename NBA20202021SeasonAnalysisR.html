Phase 3. Data Analysis Plan

In Phase 3 the documented procedure for how the two explanatory variables will be separately compared with the response variable is shown. As mentioned before the response variable which is going to be analyzed is each player’s field goal percentage for the 2020-2021 regular season. The two subsections below (3.1 and 3.2) are going to discuss the simulations for comparing the response variable with two explanatory variables: the player’s position (3.1) and the total number of regular season games a player played in (3.2).

3.1 Field Goal Percentage (Response) vs. Position

My conjecture is that players identified as guards will have a different field goal percentage than players who do not identify as guards. I believe this to be true because you could make an argument for either direction: players who are not guards typically take shots closer to the basket, thus easier to make. While guards shoot from farther away, they take those shots because they are typically better shooters.

With field goal percentage being a numeric variable and the position of a player being a binomial categorical variable, a two-mean inference procedure can be conducted as follows: The mean field goal percentage for all guards and all non-guards in the sample will be calculated. The difference between these two means is the observed test statistic.

From the 540 observational units there are 540 field goal percentages. The 540 response values will be shuffled and randomly assigned to an ‘A’ group and a ‘B’ group. The ‘A’ group will always be as large as the number of guards in the sample and the ‘B’ group will always be as large as the number of non-guards in the sample. The average field goal percentage will be calculated for each of the groups and the difference between the two groups will be calculated. Using a computer, the simulation can be repeated 999 more times for 1000 simulated test statistics.

The number of simulated test statistics which were as or more extreme as the observed test statistic will be counted. If 50 or less simulations are more extreme than the observed test statistic (p-value<0.05), this will be seen as evidence for the conjecture.

3.2 Field Goal Percentage (Response) vs. Regular Season Games Played

My conjecture is that there will be a direct association between a player’s field goal percentage and the number of games the player played in. I believe that this will be true because the players who have higher field goal percentages will play more games, because they can make shots at a more efficient rate.

Both field goal percentage as well as regular season games played are numeric variables. An XY plot can be created with 540 points. Each point is an observational unit from the sample and is plotted in the x direction by the number of games the player played in and is plotted in the y direction for the player’s field goal percentage for the season. From this xy plot a best fit line can be placed and the strength of the correlation between the two variables can be counted.

To test this conjecture a correlation will be computed, and if the correlation is greater than 0, it will be considered evidence to support the conjecture.

Before conducting the two-mean inference procedure we can state our hypotheses: let xbarGuard = the average field goal percentage for all players from the 2020-2021 season who are identified as guards let xbarNotGuard = the average field goal percentage for all players from the 2020-2021 season who are identified as non guards H0: xbarGuard = xbarNotGuard Ha: xbarGuard != (not equal to) xbarNotGuard

# Compute the observed difference in the means between the two groups
observed <- diffmean(~FieldGoalPercentage | Position, data=df3, na.rm = T)
paste("Observed difference in the means: ", round(observed, 3 ))
## [1] "Observed difference in the means:  0.057"

The observed difference in average field goal percentage between the guards and non guards is .057

set.seed(813)
trials <- 1000

# Compare the difference in means when we shuffle the Position variable - do this lots of times
null.dist <- do(trials) * diffmean(FieldGoalPercentage ~ shuffle(Position), data=df3, na.rm = T)

# Graph the histogram with the observed difference of proportions
histogram( ~ diffmean, data=null.dist, 
           main = "Sampling distribution for null hypothesis", 
           xlab = "Difference in means", 
           groups = diffmean >= observed, 
           v = observed, na.rm = T)

# Let's compute the p-value
if (observed > 0) {
  pvalue <- prop(~ diffmean >= observed, data=null.dist)
} else {
  pvalue <- prop(~ diffmean <= observed, data=null.dist)
}
paste("The two-sided p-value is ", 2 * round(pvalue,3))
## [1] "The two-sided p-value is  0"

With a p-value of approximately 0, we have evidence to support our conjecture and have evidence to reject our null hypothesis. We have evidence to believe that average field goal percentage is different for guards and non-guards.

Conclusion

After completing the statistical procedures, both generated conjectures were found to be correct. The average field goal percentage from guards and non-guards was found to be different from one another. Generally as a player’s number of games played increased, a player’s field goal percentage increased as well.

During this project I learned a lot about the practical use of the 3s method for using statistics to solve problems. I retrieved my statistics and then ran analysis on different variables independently from each other. I ran two simulations in order to test relationships between variables. Then I used strength of evidence to determine if my conjecture could be proven or not. This method definitely made presenting my information a lot easier to implement, and hopefully made my report a lot easier to read.

Looking back, instead of doing field goal percentage, I would have analyzed free throw percentage as the response variable. There are a lot of different factors which influence if a basket is made or not besides the two analyzed explanatory variables. A free throw, when the game of basketball is stopped and the only action is the shooter shooting the shot, would have been a lot easier to find a more direct relationship.
