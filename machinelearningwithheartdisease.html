Machine Learning with Heart Disease

  • This project was created during the spring semester of my senior year for STAT 375: Data Analysis for Statistical computation. The project is primarily implemented in R, with a Python script used for data importing. Various machine learning techniques, including K-Nearest Neighbors (KNN), Classification Trees, and regression using the Caret package, were employed to build predictive models.
  • The project was presented at a keystone event for capstones at the end of the semester
